# NLTK Text Processing Toolkit

A comprehensive Python class for performing essential Natural Language Processing (NLP) tasks using the Natural Language Toolkit (NLTK). This toolkit provides a simple interface to tokenization, stopword removal, stemming, lemmatization, part-of-speech tagging, and named entity recognition.

## Features

- **Sentence Tokenization** – Splits text into individual sentences.
- **Word Tokenization** – Breaks sentences into words/tokens.
- **Stop Words Removal** – Filters out common English stop words.
- **Stemming** – Reduces words to their base/root form using Porter Stemmer.
- **Lemmatization** – Converts words to their dictionary form using WordNet.
- **Part-of-Speech (POS) Tagging** – Assigns grammatical tags to each token.
- **Named Entity Recognition (NER)** – Identifies named entities (persons, organizations, locations) in text.
